# Quickstart: Patient Search

## Local Development
1. Ensure `node` >= 20 and `npm` are installed.
2. Clone the repo and run `npm install`.
3. Start the dev server: `npm run dev`.
4. Navigate to `http://localhost:5173/patients`.

## Verification Steps
- Search for 'MARTIN' in the search bar.
- Verify the table filters to show Jean-Pierre MARTIN.
- Click the 'run' icon to verify redirection to `/session/PAT-2024-089/setup`.
